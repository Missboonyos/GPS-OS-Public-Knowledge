# FAQ

**Is GPS OS a software framework?**
No. GPS OS is a problem-navigation methodology, not a code library or framework. See [00_READ_THIS_FIRST.md](../docs/00_READ_THIS_FIRST.md).

**Do I need AI to use GPS OS?**
No. AI is one possible tool within the Build phase, used only when necessary.

**Who is GPS OS for?**
Domain experts, software engineers, product managers, founders, and anyone transforming operational knowledge into software. See [01_Why.md](../docs/01_Why.md).

**Is GPS OS finished?**
No. It is an evolving body of knowledge. See [06_Roadmap.md](../docs/06_Roadmap.md).
